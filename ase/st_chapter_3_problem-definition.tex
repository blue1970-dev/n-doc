\hrefchapter{spd}{Definition Of The Security Problem}

This section firstly describes the assets that the TOE must protect and which
external entities interact with the TOE. Following that, we discuss the threats
to the TOE, the organizational security polices that have to be considered, and
the assumptions about the operating environment.
