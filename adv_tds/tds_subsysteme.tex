
\hrefchapter{sub}{Description of Subsystems}

This chapter describes the division of the TOE into its subsystems.

\subsystemchapter{sub.vpn}

\subsystemdescription{sub.vpn.desc}

The subsystem \tds{sub.vpn} provides the VPN client that is used to create VPN tunnels in the WAN.

\subsystemsfr{sub.vpn.sfr}

\generatesfrsubsystable{sub.vpn}

\subsysteminteract{sub.vpn.interact}

The subsystem \tds{sub.vpn} requires functionalities from the subsystems
\tdslink{sub.cryptsystem}.


\subsystemchapter{sub.ntpclient}

\subsystemdescription{sub.ntpclient.desc}

The subsystem \tds{sub.ntpclient} provides the NTP client that is used to
synchronize the system time with an NTP server in the WAN.

\subsystemsfr{sub.ntpclient.sfr}

\generatesfrsubsystable{sub.ntpclient}

\subsysteminteract{sub.ntpclient.interact}

\subsystemchapter{sub.selfprotect}

\subsystemdescription{sub.selfprotect.desc}

The subsystem \tds{sub.selfprotect} contains the functionalities that are used
for the self-protection of the TOE. Provided are capabilities to diagnose the
TOE with self-tests (\tdslink{mod.selfprotect.selftest} and for deleting
residual information from the TOE's memory (\tdslink{mod.selfprotect.memory}).

\subsystemsfr{sub.selfprotect.sfr}

\generatesfrsubsystable{sub.selfprotect}

\subsysteminteract{sub.selfprotect.interact}

\subsystemchapter{sub.adminsystem}

\subsystemdescription{sub.adminsystem.desc}

The subsystem \tds{sub.adminsystem} provides the administration application and
management functions of the TOE.

\subsystemsfr{sub.adminsystem.sfr}

\generatesfrsubsystable{sub.adminsystem}

\subsysteminteract{sub.adminsystem.interact}

This subsystem requires functionalities from the subsystem
\tdslink{sub.selfprotect}.

\subsystemchapter{sub.cryptsystem}

\subsystemdescription{sub.cryptsystem.desc}

The subsystem \tds{sub.cryptsystem} provides cryptographic base functions
required by other functionalities of the TOE. The base functions comprise
cryptographic algorithms, key management and random number generation.

\subsystemsfr{sub.cryptsystem.sfr}

\generatesfrsubsystable{sub.cryptsystem}

\subsysteminteract{sub.cryptsystem.interact}

\subsystemchapter{sub.tls}

The subsystem \tds{sub.cryptsystem} provides TLS connections required to connect
to the management functions in subsystem \tdslink{sub.adminsystem}.

\subsystemdescription{sub.tls.desc}

\subsystemsfr{sub.tls.sfr}

\generatesfrsubsystable{sub.tls}

\subsysteminteract{sub.tls.interact}
