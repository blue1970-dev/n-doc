\moduledescription{mod.adminsystem.webserver.desc}

The module \tds{mod.adminsystem.webserver} of subsystem
\tdslink{sub.adminsystem} provides an HTTP Server for the management interface.

\moduleprocesses{prc.adminsystem.webserver}

\moduleprocess{prc.adminsystem.webserver.httpcomm}{HTTP Communication With an
  Administrator}

This process implements the HTTP protocol.

\implementedsfr{\sfrlink{ftp_trp.1/admin}}

\moduleinterfaces{int.adminsystem.webserver}

\moduleinterfaceprovided{int.adminsystem.webserver.httpendpoint}

This interface implements the logical interface \lslanhttpmgmt{}.

\moduleinterfacerequired{int.adminsystem.mgmt.setconfig}

The interface \tdslink[fq]{int.adminsystem.mgmt.setconfig} is required for
saving configuration parameters.

\moduleinterfacerequired{int.adminsystem.mgmt.connectdisconnect}

The interface \tdslink[fq]{int.adminsystem.mgmt.connectdisconnect} is required
by the HTTP server for opening/closing connections to the VPN concentrator.
